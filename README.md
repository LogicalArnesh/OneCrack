# OneCrack Test Portal

A professional academic testing platform built with Next.js, Firebase, and Genkit AI.

## 🚀 How to push to GitHub

Follow these steps in your terminal to push this code to your repository:

1. **Initialize Git**:
   ```bash
   git init
   ```

2. **Add all files**:
   ```bash
   git add .
   ```

3. **Commit the changes**:
   ```bash
   git commit -m "Initial commit: OneCrack Test Portal"
   ```

4. **Link to GitHub**:
   ```bash
   git remote add origin https://github.com/LogicalArnesh/OneCrack.git
   ```

5. **Push to Main**:
   ```bash
   git branch -M main
   git push -u origin main
   ```

## Features
- **Secure Portal**: Firebase Auth & Firestore based secure login.
- **AI Question Importer**: Admin can upload PDFs to extract questions via Genkit.
- **AI Personal Mentor**: Personalized study plans generated by Gemini based on your scores.
- **Professional Analytics**: High-fidelity result dashboards with Recharts.
